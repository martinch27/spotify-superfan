# Spotify Super-Fan — кликабельный прототип

Лендинг-прототип для презентации идеи **Super-Fan tier** — нового конверсионного слоя для Spotify, который монетизирует самых преданных слушателей артиста.

## О чём проект

Spotify уже владеет данными о том, кто кого и сколько слушает (Super Listeners — 2% MAL артиста). Super-Fan tier использует эту структурную перевагу: точечный push на топ-1% слушателей конкретного артиста с предложением подписаться напрямую на эксклюзивный контент.

В прототипе:

1. Открывается интерфейс Spotify с библиотекой Martin
2. Через секунду — поп-ап «Ты в топ 1% фанатов Playboi Carti» с предложением стать Super-Fan
3. CTA → экран выбора уровня (Essential $4,99 / Insider $19,99 / Inner Circle $99,99)
4. → Чекаут с Apple Pay
5. → Экран успешной подписки с эксклюзивными перками

## Локальный запуск

Открой `index.html` в браузере — это статичный сайт без билда.

## Структура

- `index.html` — все 4 экрана (home, tiers, checkout, success) и поп-ап
- `styles.css` — дизайн в стиле Spotify mobile
- `app.js` — навигация между экранами и логика поп-апа

## Деплой

Хостится на GitHub Pages. После пуша в `main` сайт автоматически собирается.
