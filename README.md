# Spotify SuperFan — клікабельний прототип

Лендинг-прототип для презентації ідеї **SuperFan tier** — нового конверсійного шару для Spotify, який монетизує найвідданіших слухачів.

## Про що проєкт

Spotify уже володіє даними про те, хто кого і скільки слухає (Super Listeners — 2% MAL). SuperFan tier використовує цю структурну перевагу: точковий push на топ-1% слухачів з пропозицією підписатись на ексклюзивний контент.

У прототипі:

1. Відкривається інтерфейс Spotify з бібліотекою Мартіна
2. Через секунду — поп-ап «Ти у топ 1% слухачів Плейліста дня 1» з пропозицією стати SuperFan
3. CTA → екран вибору рівня (Essential $4,99 / Insider $19,99 / Inner Circle $99,99)
4. → Чекаут з Apple Pay
5. → Екран успішної підписки з ексклюзивними перками

## Локальний запуск

Відкрий `index.html` у браузері — це статичний сайт без білду.

## Структура

- `index.html` — усі 4 екрани (home, tiers, checkout, success) та поп-ап
- `styles.css` — дизайн у стилі Spotify mobile
- `app.js` — навігація між екранами та логіка поп-апа
- `qr.html` + `qr.png` — QR-код для проєктора, веде на live-демо

## Деплой

Хоститься на GitHub Pages. Після пушу в `main` сайт автоматично пересобирається.
